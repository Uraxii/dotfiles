# Uraxii Dotfiles

## Requirements

### Git

Ensure Git is installed to Pull/Push changes to these files.

```
$ sudo pacman -S git
```

### Stow

Install GNU Stow

```
$ sudo pacmac -S stow
```

Create symbolic links to restore dotfiles.

```
$ cd ~/dotfiles
$ stow .
```

### Neovim

Install Neovim

```
$ sudo pacman -S neovim
```

#### Dependencies

- node
- fzf
- gcc
- go-lang
- unzip

```
$ sudo pacman-Syu node, unzip, gcc, fzf, go-lang
```
