# Shell

Zsh + oh-my-posh prompt + ghostty terminal.

## .zshrc

### Purpose

Interactive Zsh setup. Initialises oh-my-posh, key bindings, aliases,
and tool integrations (nvm, zoxide, opencode).

### Key files

- `.zshrc` — main config.
- `.zprofile` — login env (currently exports `QT_QPA_PLATFORMTHEME=qt6ct`).

### Keybindings & UX

| Sequence | Bind | Action |
|----------|------|--------|
| `^[[H` | Home | beginning-of-line |
| `^[[F` | End | end-of-line |
| `^[[3~` | Del | delete-char |
| `^[[1;5C` | Ctrl+Right | forward-word |
| `^[[1;5D` | Ctrl+Left | backward-word |

Alias: `logout` -> `swaymsg exit` (only when `$SWAYSOCK` set).

Path: prepends `$HOME/.local/bin` and `/home/nikki/.opencode/bin`.

### Theming integration

Sources oh-my-posh from runtime
`~/.config/omp/uraxii_atomic.omp.toml` (generated by `set-theme.sh`
from the `.tmpl` + theme `omp-colors`). See
[theming.md](theming.md).

### External dependencies

`zsh`, `oh-my-posh`, `zoxide`, `nvm` (optional), `opencode` (optional).

## oh-my-posh

### Purpose

Cross-shell prompt. Theme-templated TOML.

### Key files

- `.config/omp/uraxii_atomic.omp.toml.tmpl` — live template.
  Placeholder syntax: `##NAME##` (TOML-safe; avoids Go-template
  collision with `{{...}}`).
- `.config/omp/uraxii_atomic.omp.toml` — runtime, generated.
- `.config/omp/cert.omp.toml`, `.config/omp/atomic.omp.json` —
  reference / legacy.

### Theming integration

`set-theme.sh` sources `themes/<name>/data/omp-colors` (shell-style
`KEY=value`) and substitutes the following keys into the template:
`PRIMARY`, `PATH_BG`, `PATH_FG`, `GIT_BG`, `GIT_FG`, `GIT_DIRTY`,
`GIT_AHEAD`, `MUTED`, `OS_BG`, `FG_DARK`, `FG`, `ERROR`.

### External dependencies

`oh-my-posh`.

## ghostty

### Purpose

GPU-accelerated terminal emulator. Set as `$term` in sway.

### Key files

- `.config/ghostty/config`.

Active settings: `theme = "Kanagawa Dragon"`,
`background-opacity = 0.8`, `font-family = 0xProto Nerd Font Mono`,
`font-size = 14`. Sway also applies a per-window opacity rule (see
`sway/modules/windows`).

### Theming integration

Currently independent of the sway theming pipeline (ghostty has its
own theme key). Add a templated theme key here if/when ghostty enters
the pipeline.

### External dependencies

`ghostty`, `0xProto Nerd Font` (`nerd-fonts._0xproto` in `home.nix`).
