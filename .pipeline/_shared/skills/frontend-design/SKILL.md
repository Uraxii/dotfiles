Build distinctive prod-grade frontend. No AI slop. Real working code, exceptional aesthetics.

## Design Thinking

Before code → commit BOLD aesthetic direction:
- **Purpose**: Problem solved? Who uses?
- **Tone**: Pick extreme — brutalist, maximalist, retro-futuristic, organic, luxury, editorial, art deco, industrial, etc. True to direction.
- **Constraints**: Framework, perf, a11y reqs.
- **Differentiation**: What makes UNFORGETTABLE?

**CRITICAL**: Clear conceptual direction → precise execution. Bold maximalism + refined minimalism both work — intentionality > intensity.

Impl working code (HTML/CSS/JS, React, Vue, etc.): prod-grade, visually striking, cohesive aesthetic POV, meticulous detail.

## Aesthetics

- **Typography**: Beautiful + unique fonts. NO generic (Arial/Inter/Roboto/system). Distinctive display font + refined body font.
- **Color**: Cohesive palette via CSS vars. Dominant colors + sharp accents > timid even distribution.
- **Motion**: CSS-only for HTML, Motion lib for React. Focus high-impact: orchestrated page load w/ staggered `animation-delay` > scattered micro-interactions. Scroll-trigger + surprising hover states.
- **Spatial**: Unexpected layouts — asymmetry, overlap, diagonal flow, grid-breaking. Generous whitespace OR controlled density.
- **Backgrounds**: Atmosphere + depth, not solid colors. Gradient meshes, noise textures, geometric patterns, layered transparencies, dramatic shadows, decorative borders, grain overlays.

NEVER: generic AI aesthetics — overused fonts, cliché purple gradients on white, predictable layouts, cookie-cutter patterns.

Interpret creatively. No two designs same. Vary light/dark, fonts, aesthetics. NEVER converge on common choices across generations.

Match impl complexity → aesthetic vision. Maximalist = elaborate code + extensive animation. Minimalist = restraint + precision spacing/type/subtlety.

Push boundaries. Commit fully to distinctive vision.
