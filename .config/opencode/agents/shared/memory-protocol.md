# Memory Protocol

After work → write memories. `<ROLE>` = your agent name.

| Scope | Path |
|-------|------|
| Universal lessons | `~/.config/opencode/memory/<ROLE>-memory.md` |
| Project domain | `<project>/.opencode/memory/<ROLE>-memory.md` |
| Cross-cutting (global) | `~/.config/opencode/memory/core-memory.md` |
| Cross-cutting (project) | `<project>/.opencode/memory/core-memory.md` |

Cross-cutting → only if pattern spans multiple roles.
