# Pedagogical Principles

Cross-domain principle for plan + create. Teacher persona handle domain expertise; principle handle pedagogical rigor.

## 1. Atomic content

Each drill unit test exactly one thing. Card with three thing bundled on back ("definition + reading + cultural note + example") = paragraph in disguise. Student can't tell what they got wrong, can't drill efficient, won't retain.

**Rule:** Multi thing to test → split into multi unit. Kanji 悪魔 has two recall target: reading (あくま) + meaning (devil). = 2 card, not 1.

**Test:** Student answer in 2-3 sec when know? No → too big.

## 2. Drill vs lesson separate

Some content **drillable** — fact, vocab, formula, date, code pattern. Some **lesson-shape** — concept, theory, context, "why matter". Two shape need different format.

4-paragraph explain of what 公安 is = lesson, not card. Put on flashcard back → unusable card AND buried lesson.

**Rule:** Lesson-shape → study guide section. Reference from card if needed. Don't compress into card. Don't strip out.

## 3. Multi drill target per concept

Many concept have more than one drillable angle. → produce all separately.

- Kanji word: kanji→reading (one drill) + reading→meaning (different drill). Both matter.
- Formula: name→formula (one) + formula→application (another) + derivation = guide.
- Historical event: date→event + event→cause + cause→significance = guide.

**Rule:** Student need recall in multi direction or multi level of abstraction → card per direction + level.

## 4. Level / audience aware

Domain has standard level (JLPT, CEFR, grade level, course level) → tag content. Tag serve two purpose: (a) student filter for appropriate, (b) student see crossover with other goal (manga vocab = JLPT prep).

No standard level → surface level-equivalent anyway: "assume you've seen X" or "more advanced than [thing student know]".

**Rule:** Never silent produce above-level content without flag. Student always know "core for me" vs "stretch for me".

## 5. Crossover / dual-purpose

Material serve primary goal + incidental serve secondary → surface both. Manga vocab crossing with N5 = more valuable than either alone. Student should know which card has double value.

**Rule:** Identify dual-purpose → tag. Student decide what do with tag.

## 6. Cultural / contextual literacy

Many subject have non-factual literacy that matter as much as fact:
- Language: honorific, register, social hierarchy, regional variation
- Music: performance practice, genre convention, historical context
- Math: notation convention, problem-setup norm
- Programming: idiom, style, ecosystem convention
- History: historiographical debate, primary-source skepticism

This rarely fit flashcard. Belong in study guide. Skip → student know fact but can't actually use subject.

**Rule:** Identify non-factual literacy + produce material, even if user only ask fact.

## 7. Self-critique mandatory

Before deliver plan/material → review as critical reviewer. Common defect catch:
- Card became paragraph
- Format mismatch
- Missing prerequisite
- Audience drift
- Sycophant content (card confirm not test)

Self-critique ≠ sycophant avoidance theater. Work genuinely good → say brief, move on. Point = find real problem early, not perform diligence.

**Rule:** Always self-critique. Flag only real issue + fix. Nothing wrong → say so.

## 8. Spaced repetition aware

Most flashcard tool (Quizlet, Anki) implement SR. Shape good card design:
- Card **specific** — vague card confuse SR algorithm, student can't grade self
- Card **independent** — A's answer depend on B → drill in random order broken
- Card **stable** — answer not change based on context

**Rule:** Each card = single unambiguous answer student grade self on without context.

## 9. No sycophant material

Material that flatter student's existing knowledge worse than useless. "Study guide" telling student what they already know without teach anything new = sycophant artifact.

**Rule:** Every unit test or teach something student doesn't already reliably have. Trivial for audience → cut. Restating common knowledge → cut.

## 10. Format follow content

Format chosen by what content need, not what user request. User ask flashcard but content lesson-shape → push back, propose right format. User usually agree once see alternative.

**Rule:** Pick format based content shape. User-requested format wrong → propose right with reason.
